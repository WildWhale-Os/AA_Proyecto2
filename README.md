# Proyecto 2 Analisis de Algoritmos

## Primero que todo
Crear una carpeta build en el directorio root del Proyecto.

  ```bash
  $ mkdir build/
  ```

## Building
Para construir el archivo debe ejecutar, estos dos commandos estando
en el directorio root del Proyecto

 ```bash
 $ cmake .
 ```

 ```bash
 $ cmake --build .
 ```
